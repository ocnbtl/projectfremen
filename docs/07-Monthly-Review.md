# Monthly Review

Use this once per month (recommended: first Sunday of the month).

Goal: check business health, clean up operations, and set clear priorities for the next month.

Target duration: under 60 minutes.

## Monthly Checklist

### 1) Confirm Last Month Snapshot (10-15 minutes)

1. Review KPI values from the final week of last month.
2. Record what improved, what declined, and what stayed flat.
3. List the top 3 wins.
4. List the top 3 misses.

### 2) Review Each Project (15-25 minutes)

1. Project Fremen (Unigentamos):
   - documentation coverage trend
   - blocker trend
   - dashboard reliability and usability
2. Project Iceflake (pngwn):
   - waitlist growth trend
   - impression trend
   - sentry error trend
   - inbox load trend
3. Project Pint (Diyesu Decor):
   - publishing consistency (pins/blogs)
   - outbound click trend
   - impression trend
   - newsletter growth trend
   - inbox load trend

### 3) Finance and Admin Reality Check (15 minutes)

1. Confirm monthly expenses were captured.
2. Confirm any recurring subscriptions still make sense.
3. Confirm business account, tax, and filing reminders are still current.
4. Confirm no missed renewals or compliance deadlines.

### 4) Systems and Security Check (10 minutes)

1. Confirm GitHub access is correct (only needed accounts have write access).
2. Confirm tokens/secrets are stored only in env or secure managers.
3. Confirm no sensitive values were committed to public repos.
4. Confirm dashboard auth still works and password policy is still acceptable for MVP.

### 5) Documentation and Workflow Cleanup (10-20 minutes)

1. Archive stale tasks that no longer matter.
2. Update docs that are now inaccurate.
3. Add missing KPI definitions or data-source notes.
4. Note where AI Agent can replace manual updates next month.

### 6) Set Next Month Plan (15 minutes)

1. Pick 3 monthly outcomes (max 3).
2. Assign owner for each outcome (`Me`, `AI Agent`, `Auto`).
3. Pick 1 KPI target per project for next month.
4. Add or update tasks in dashboard.
